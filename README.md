# Todo-App 
지인들과 일정을 공유하고 일일 완료율을 확인해보세요



## 프로젝트 목표
지인과 TODO를 공유하며 서로 자극을 주기  위해 매번 TODO 일정을 캡쳐하여 공유한다. 이런 부분에서 번거로움을 느껴 해소 하고자 사용자들끼리 방을 만들어 서로의 TODO를 확인할 수 있게 도와주기 위함이다.
